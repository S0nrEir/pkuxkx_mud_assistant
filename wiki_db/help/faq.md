07：人物属性参数是怎么确定的？

属性参数分先天与后天，包括膂力、悟性、根骨、身法、福缘、容貌、灵性共七个。每个人出生时的膂力、悟性、根骨、身法总和为 80，灵性为30，这些都是先天参数。

后天参数是指在先天的基础上通过勤学苦练提高自己的某些技能等级而达到新的水平，但是这不能改变自己的先天属性。

技能与属性对应如下（技能长10级，对应属性长1）：基本空手武功（unarmed、hand、strike、cuff、finger、claw等）长膂力，读书识字长悟性，基本内功长根骨，基本轻功长身法。

08：有什么办法提高先天属性吗？

新人在24岁前完成晋阳洗澡任务可以提高1-3点根骨。特内高于301级时，完成初通小周天来打通任督二脉可以提高各种先天属性 1-2 点，归云庄的池塘洗澡可改变自己的容貌（高福缘加容貌，低福缘减容貌），到武庙捐献可以提高自己的福缘（上限最高可捐到30）。莫高窟冥想可提高共计6点先天，其中灵性大概率，其他小概率。完成一灯大师的任务可以提高先天悟性2点，先天根骨1点。新人里程碑任务有极小概率获得1点相应先天。过江过河时身上偶尔出现的神秘线索到相应地点（同名房间只有一个有效）获得并阅读武学心得有极小概率获得1点先天奖励。洛阳中心广场晓晓处每年注册生日当月可领取生日蛋糕，吃了大概率获得福缘/容貌/灵性1点，小概率获得前四项先天一点。此外，江湖频道出现的部分事件会掉落仙丹（神力丸、美容丸之类的），拣到吃了有一定概率可获得相应先天奖励一点，还有看到凤凰起舞等事件也有一定概率可获得先天奖励。

09：怎么提高自己的武功等级？

跟师父学习（learn），需要消耗潜能，当你的技能超过师父的技能后，学习就转变为切磋，潜能消耗大幅提高，切磋也分有效切磋与无效切磋，详见技能学习。读书（study），需要会读书写字(literate)，不同的书可能有不同的要求；特殊技能通过练习（practice）提高，需要基本武功等级不低于特殊武功等级（有些武功不能练习）；基本技能通过领悟（lingwu）提高，需要过少林罗汉阵，在剑心居、各门派领悟地点和帮派领悟地点进行，领悟和学习可以**交叉进行，逐级提高技能**；特殊内功通过修炼(xiulian)提高；所有的基本技能都可以在侠客岛上读石壁提高。部分特殊技能只能通过特殊方式获取和提升，如岳家枪、乾坤大挪移、斗转星移等。此外，萧峰、韩世忠任务有一定概率奖励一级技能（萧峰困难模式为两级，占卜等少部分技能设置为不能奖励，福缘低奖励技能概率大）。张三丰讲解、武学之道的闭关和讲解也都会增加技能小点（特殊内功及部分技能被设定为不增加）。

10：为什么我不能跟我师傅学武功？

通常是你输错了你师傅的id，请用id here看看你师傅的id，或者你师傅不会这种技能，可用skills <师傅id> 或cha -learn <师傅ID> 查看师傅所掌握的技能。用 learn <师傅id> for <技能> <次数>。

11：什么是武功的有效等级？

你输入 enable 看到的等级就是你的武功有效等级。有效等级为：

```
基本武功等级/2 + 特殊武功等级
```

12：各种技能的最高等级有什么限制？

知识类技能的等级没有限制，其他技能等级的理论上限为:

```
(技能等级的三次方) < exp×10
```

而实际上限受到先天悟性的影响，技能等级的实际上限为:

```
理论上限 × ( 1 + (先天悟性 - 30) × 0.5% )
```

也就是说，先天悟性 = 30，实际上限等于理论上限；先天悟性 < 30，实际上限小于理论上限；先天悟性 > 30，实际上限大于理论上限。

13：人物精神与气血值与什么有关

刚出来的新人年龄为 14 岁，精神与气血都为 100，14－30 岁每年精神与气血都会增加。精神增加为自己的先天悟性，气血增加为自己的先天根骨。30－60岁精神与气血不变，过了60以后每年精神与气血都会减少。

最大精力增加 4 点，精神会增加一点，最大内力增加 4 点，气血会增加一点。

14：怎么长最大精力与最大内力？

吐纳（tuna）可以长最大精力，新手在ct（pick guo）可以拿到新手礼物，长 1000 经验和 40 最大精力。

打坐（dz 或 dazuo）可以长最大内力，ct往上的赏月台可以找绿萼要月饼，甜月饼长50最大内力，咸月饼长50最大精力（两者只能要其一）。

另外有些药物可以长最大精力与最大内力。吃药长最大内力速度快，但通过打坐方式可以得到更高的内力精纯度。

15：最大精力上限与什么有关？最大内力上限与什么有关？

通过吐纳（tuna）可以达到的最大精力上限＝当前有效内功等级×10。

通过打坐（dz和 dazuo）可以达到的最大内力上限＝当前有效内功等级×10。

通过服用药物等方法提高则没有限制，但是玩家登陆之后会检测最大内力上限，超过有效内功等级×10的部分会被**清除**。

16：什么地方是室外？什么地方是室内？

有天色描述的地方是室外，没有的是室内。

17：为什么我打架时用的是拳头，没有招式？

空手武功有多种（cuff、hand、finger、claw、strike、unarmed），需要先prepare <技能名称> 系统才能确定你用哪种武功，否则就是用的拳头。如果你的两种武功可以互备，可以 prepare <武功1> <武功2>。

18：怎么长道德？

道德有正有负。如果想改变道德属性，可以先去少林的达摩洞面壁，道德就会变为零。杀了比自己经验高的人，便会得到一定的道德。如果想得正道德，就去杀那些负道德的人。武当首侠宋远桥的不孝之子宋青书因杀了其师叔莫声谷，逃到了峨嵋，躲在万年庵中，如果夺下他的佩剑，把佩剑交给送远桥，可以涨道德5000。日月神教的农夫是华山派的奸细，ask about 华山派，他就会原形闭露，杀了他可以得到黑木崖地图，交与岳不群可以涨道德5000。将地图交给童百熊则会减5000道德（日月教弟子长门忠）。另外，三阴绝户掌的绝户和马屁神功的求饶若成功也会降道德。到岳王墓凭吊一次长100道德，杀秦侩会清除自己的负道德，同时增加 1万 道德。在扬州城外的随机村落击杀村民会减少大量道德。

19：怎么进少林寺？

外派如果未过十八罗汉大阵的要进少林需持少林英雄令，直接给扬州城南大街的托钵僧 1 gold可以拿到少林英雄令。女客除了需要英雄令外，还要打晕守山门的虚通、虚明，才能进，或者坐车去少林绕过门口的虚明两人。当然你如果过阵了就随便进了。在少林寺外的松林掉进陷阱后会被抓进少林监狱，贿赂狱卒10两银子可以到五行洞，从五行洞出来的话可以到少林寺内。当然，找人把自己背进去也是一个办法。

20：怎么加武林声望？

做信件任务（非慕容弟子），护卫（非灵鹫弟子），护镖，破阵任务，保卫襄阳，长安爵位任务（500K经验之前）都可以加声望。

21：做门忠任务为什么告诉我说“经验最好再高些”，需要多少经验才行？

需要 经验*10 > 门忠^2 ，做门忠任务才会得到门忠奖励。

22：怎么才能上无量山？

基本轻功 80 级以上可以爬上无量山，回来要走铁索桥，需要基本轻功 100级，否则有可能摔死。

23：怎么赚钱

捡东西卖，打npc抢他们身上的钱，跟老玩家要，抄经，详见help task-newbie

24：物品和npc什么时候会更新？

输入 uptime 看看当前时间，一般在0，15，30，45分钟的时候会更新。

25：如何打通任督二脉

特殊内功高于300后dz就有机会打通，福缘和先天根骨高对打通有好处。特殊内功等级越高越有利于打通二脉。打通二脉后各种先天属性随机增加1~2点。

26：怎么领悟武功？

首先要过十八罗汉阵，然后到门派领悟点/剑心居领悟。例如领悟基本剑法，需要特殊剑法等级大于等于基本剑法，输入 lingwu sword <次数> 即可。

27：过十八罗汉阵需要多少经验？

建议30w 经验以上，内力高一些，不停吸气（yun recover)即可。

28：为什么华山派的劳德诺会追杀我？

你身上带着《紫霞秘笈》就会遭到劳德诺的追杀。

29：我掉进杀手帮监狱了，怎么出去？

杀气太高的话进杀手帮广场会掉进监狱，出去的办法是(qiao wall，得到锯子，saw)，就可以出去了。

30：为什么kill有的npc时出现“你突然看到一把长剑已经架到了你的脖子上”？

你杀的是杀手帮的密探，这些npc有全真李余，丐帮小糊涂，襄阳郭没，大理孙拳等，当你经验小于 2M 时对他们下kill，hit，fight指令时会出现这种情况，并且你会立即晕倒。

31：为什么官兵会通缉我？通缉了我该怎么办？

如果你对官兵和武将下了 fight 和 kill 指令，就会被通缉。如果非得打他又不想被通缉，就用 hit 。通缉过一段时间就会取消，所以你老实的呆着就过一段时间就没事了。

32：精神受伤了该怎么办？可以自己用内功治疗吗？

目前还不能自己治，解决办法有多个。慢慢等他会自己恢复，隔一分钟左右吃个养精丹，可以加快恢复速度。去浴室洗澡可以完全恢复，吃天香断玉膏也可以治好，或者找人用一阳指帮你疗伤。

36：为什么我读不了《射雕英雄传》或者《天龙八部》？

因为这两本书是小说，你要用 read 指令而不是 du 或者 study。另外天龙八部要求基本刀法和剑法都达到 100 级才能看的懂。

37：怎么做任务？如何知道我当前有什么任务？

各门派都有分派门忠任务的npc，请查看门派help。ask <xx> about quest，如果让你找物品回来，找到交给npc就可以了，其他的完成后需要回来复命,就是回去再ask <xx> about quest就可以了。其他任务请看wiki主页游戏篇的任务部分或是help tasks。查询自己当前任务请用questjobquery指令。

38：什么是福缘？如何涨福缘？

福缘是几大基本属性之一，对游戏中的一些方面有影响。多数情况下福缘高比较有利，但也有些场合福缘低点更好，比如护镖时，如果福缘太高，则劫匪出现较少，完成任务虽然更容易，但奖励也相应减少。

福缘的初始值范围是10～30，出新手村前可以通过changegift自行设置为10～30之间的数值。若角色福缘低于30，可以通过武庙捐献(juanxian)来增长，捐献的规则为：第一次捐献满1000两黄金或等值银票福缘增长一点，要再涨一点福缘就需要再捐献2000两黄金或等值银票，涨第三点福缘需要再捐献3000两，以此类推，通过捐献涨n点福缘则需要总共捐献1000*(1+2+…+n)两黄金。捐献可以分次分批，累计捐献金额一直有效。下线不会丢失。通过捐献至多增长至30福缘。

其他增长福缘的方式还包括各种随机先天奖励，例如打通小周天或提升人物等级。吃武林奇遇掉下的福缘丸,（可能已经不存在）过年过节时发的极品饺子等礼品，但几率很小。

39: 过阵

**条件：**所有学会的技能等级达到80级。（知识类技能除外）

**步骤：**

-  首先进入少林寺。扬州广场s;s找托钵僧，give seng 1 gold。他会给你个令，拿着这个令牌可以进入少林寺（过了罗汉阵之后就可以自由进入少林寺了）**注：**此时最好穿上一身新手套装，增加防御力,去荣宝斋买个shield，如果血量较低的话还可以带上几瓶药（药在平一指处购买）**附：**新手护甲装拿法——扬州中央广场（down;out;draw boots;draw head;draw cloth;draw armor;draw surcoat;enter;remove all;wear all;wield all）也可以拿武器。

-  来到少林寺般若堂五部ask dashi about 罗汉大阵，然后向东走4步（do 4 e），来到罗汉堂五部 ask dashi about 罗汉大阵。

-  问完两位大师以后来到中间的比武场（向西走两步方丈楼下面）等待过剧情。

-  开打，被打的时候可以看情况吸气（yun recover），埃够18个僧人每人一下后就算完成了。这时会看到江湖频道公布成功的消息。从此你就可以领悟（lingwu）基础武功了。

40：车船通（行旅通）是什么？去哪购买？

车船通在2022年8月更名为行旅通，类似于现实中的公交充值卡，充值后坐车坐船，住店直接扣费，不需要再手动给钱。在扬州杂货铺购买，有两种不同面额（20金或100金）的行旅通，购买后输入指令addvalue进行充值。门派例银也可以用于充值行旅通。

41：如何解决吃喝问题？

每个门派都设有拿吃喝的地方，在扬州中心也有路径去免费吃喝的地方，具体方式请help node，然后去免费吃喝的地方。如果有一定资金基础，可以在扬州醉仙楼买干粮，洛阳望星楼买各种汤，买葫芦去扬州茶庄装水也是可以的，这样就能随身携带，边走边吃喝。

42：如何买药吃？

各地药铺出售养精丹（恢复精神上限）和金创药（恢复气血上限）。**注意请养成买药输入完整ID（如buy jinchuang yao)的习惯**，因为药铺时常会出售假药”噬嗑丹“，服用该假药后会丧失语言能力（该状态无法说话，包括无法接任务），需服用清音丹（价值1500金）才能恢复!

43：为什么进出副本个人状态会不一样，有时甚至退出剑心居就晕倒？

这种情况通常是因为账号的数值发生了很大变化，但是主站状态刷新滞后（一般需要重新登陆来刷新）。在此状态下，当玩家进入副本时，副本中的状态相当于重新登陆刷新后的，再回到主站时又会变回刷新前的，此时如果某些数值变化是根据刷新后的状态计算的，就可能会造成越界导致晕倒甚至死亡。
解决办法：在主站大退，即在主站quit，然后再登陆，刷新主站状态即可。注意：paotin由于重载了quit，需要打大写的QUIT才能真正退出，否则只是断线重连，不会刷新状态。
