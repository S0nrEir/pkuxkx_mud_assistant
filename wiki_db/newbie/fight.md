战斗系统

北侠的战斗系统是半回合制，每一回合的计算是在服务器上依据心跳时间自动进行的。如果你这一回合处于繁忙(busy)状态，那该回合就不能进行攻击。部分技能可以在一个回合中进行多次普通攻击。还有一些武功具有多段连招。战斗中双方可以使用能使对方无法攻击，进入繁忙状态的绝招（perform,缩写为pfm)，通常称之为busy。

武功系统

通用功夫

通用的基本功夫有**内功(force)、轻功(dodge)和招架(parry)**。

通常来说，内功等级直接影响角色的内力、精力上限。轻功影响躲闪值。而招架则影响招架值。

提升基本内功技能会提高后天根骨，每提升10级，提高1点后天根骨。

提升基本轻功技能会提高后天身法，每提升10级，提高1点后天身法。

对应的特殊功夫是特殊内功、特殊轻功与特殊招架。

绝大部分的拳脚或兵器功夫都能激发为招架，兵器功夫的招架只在手握对应兵器时生效。

极小部分的拳脚或兵器功夫可以激发为轻功（例如辟邪剑法、无影幻腿、金蛇游身掌和太祖盘龙棍）。

特殊轻功的详细介绍参见：轻功汇总

拳脚功夫

北侠有如下基本空手技能：

**拳脚 (unarmed)、拳法 (cuff)、腿法 (leg)、爪法 (claw)、掌法 (strike)、手法 (hand)、指法 (finger)**

提升基本空手类技能会提高后天膂力，每提升10级，提高1点后天膂力。只取**等级最高**的类别，学习多种空手技能无额外后天膂力加成。

特殊技能和基本技能对应，每类功夫都有对应的特殊功夫，如：降龙十八掌对应基本掌法(strike)，一阳指对应基本指法(finger)，天山折梅手对应基本手法(hand)等等。

而基本拳脚 (unarmed) 则可以用于激发任意一种空手特殊技能。

**verify 技能id** 可以知道该特殊技能的各类数据，包括该技能可以激发的基本技能。

verify展示的表格中，**基础伤害**在一定程度上影响外功伤害。**力量伤害**在一定程度上影响内攻伤害。**攻击速度**影响普通攻击的频率。

空手功夫的普通攻击通常在空着右手的时候才能使用。空手功夫激发（enable/jifa）并准备指定（prepare/bei）后才能使用该空手技能进行战斗。

**若你没有指定所要用的拳脚技能（prepare/bei）那么只会自动使用无招式的基本拳脚进行战斗**。

不同于普通攻击，大部分空手绝招(pfm)可以在空出任意一只手的时候使用，即使你右手拿着武器

兵器功夫

北侠有如下常见兵器技能：

**剑法 (sword)、刀法 (blade)、鞭法 (whip)、匕法 (dagger)、锤法 (hammer)、枪法 (spear)、杖法 (staff)**

另有一些少见的兵器技能：

戟法 (halberd)、棍法 (club) 、棒法(stick)、斧法 (axe)

特殊兵器技能也与基本技能对应，每类功夫有对应的特殊功夫。

不同于空手绝招，角色通常只能使用**右手武器**的绝招(pfm)。

武器

当角色装备了武器在手里，战斗中会使用对应的武器功夫。同样，需要和jifa激发配合，激发了特殊功夫才会在战斗中使用特殊功夫，不然只会使用基本技能进行普通攻击。

装备武器的命令

```
wield <武器ID>
wi <武器ID>
```

还可以在武器ID后面加数字表示同样ID的第几把武器。比如，你身上有两把长剑(changjian)，你想装备第二把：

```
wield changjian 2
```

你也可以指定是左手还是右手拿武器，例如：
wield blade at right 右手拿刀
wield blade 2 at left 左手拿第二把刀

放下武器的命令(unwield,缩写uw)：

```
unwield <武器ID>
unwield all    放下所有武器
unwield right  放下右手武器
uw <武器ID> 缩写
```

此命令里的武器ID也可以加数字，数字的规则和顺序同上，使用right （右手）或者left （左手）可以选择放下哪个手的武器。

武器的属性：

-  **武器伤害：**武器装备后，括号里的数字是武器伤害：例如**长剑(+20)，武士刀(+180)** 数字越高表示武器伤害越高，影响兵器功夫造成的伤害。

-  **随机武器的可塑性：**也是老玩家经常说的o，比如：3o ，4o 就是指武器的可塑性为3或4（可理解为宝石槽位），可以用来镶嵌宝石，来提高武器的效果。详见宝石系统

-  武器当前在锻造上可以镶嵌3孔的，4孔的武器虽说更好但是无法镶嵌，参看宝石系统的锻造镶嵌部分。

-  **武器耐久度：**使用武器战斗时会降低，一般武器降低为0后会损坏，可以找发呆室的爱爱修理。加载过的随机武器，耐久降低为0后无法使用，需要去各地铁匠铺（扬州除外）修理。指令为fix <武器id>。

防具

当你用 i 命令查看自己的行囊的时候，身上可穿的各种部位就是防具，如：头，披风，衣服，甲，鞋，盾，等等

手持盾会占用左手槽位。虽说它自己有个自己的位置，但一般来说，不可能把盾挂在胳膊，或者腰上，所以用左手拿盾。自然就无法装备左手武器了，也不能用左手发空手pfm。“盾击”特技为玄武职业技能。盾牌一般属于防具。

防具相关命令如下：

-  穿防具用wear命令，如：wear armor（穿铁甲）

-  卸下防具remove命令，如：remove armor （卸下铁甲）

-  防具属性：

-  护甲数值(+xxx)并不是很重要，因为防具的特点不是降低伤害，而是降低对最大气血的伤害，对当前气血，基本没什么防御作用，被砍了肯定不会还原到没被砍之前，所以是最大值的防御作用。

-  可塑性同武器。

气势系统

气势是战斗中使用某些绝招的必要条件。普通攻击增加气势，绝招消耗气势，非战斗状态不使用气势。

增加气势

普通攻击会增加气势（连招除外）。这里普及战斗系统的一个概念，战斗系统命中分为两节命中判定，第一次判定是是否运行你在本心跳发招及发多少招，这个和对方是否处于昏迷状态以及敌我双方的战斗身法都有关系，第二次判定是本次攻击是否被招架或者躲避。

**以下几种情况会导致无法获得气势：**

-  目标晕倒

-  你处于无法增加气势的状态中

**以下情况会减少你的行动力收益：**

-  目标处于busy状态

-  你使用的武功处于互备状态

-  你激活了左右互博（相比互备差异在于还有一个50%几率判定）

-  该npc被标明为群攻npc

正常情况下每秒你发出的第一次攻击增加气势更多，第二次攻击气势增加减半，第三次及以后攻击不但减半还有50%概率无气势收益。

气势维持

仅战斗状态下能够维持气势，当你退出战斗状态，气势快速衰减。

气势消耗

仅perform发出来的招式会消耗气势，仅以下几类招式会和气势有关：

busy、伤害类招式、一招晕
不同招式需要气势有所不同，对于busy来说发招需要一定气势门槛，但是发招本身不会消耗任何气势，无论使用成功与否，你将会在未来15秒内无法增加气势。

伤害类招式中单体伤害绝招需要的气势比群体伤害绝招少，但是都或多或少要消耗气势。

一招晕需要非常多的气势，每次发招无论成功与否，消耗掉一半气势，并且在未来30秒内无法获得任何气势。

战前准备

激发jifa

```
jifa 基本武功 特殊武功
如： jifa force beiming-shengong;
取消： jifa force none;
```

作用就是把某一个特殊武功和基本武功链接起来，这样使用时候就可以使用指定的特殊功夫。每个人都可以学很多功夫，比如，太极神功，九阴神功，北冥神功，九阳神功，等等。
每种功夫都有自己的特殊功效，你必须指定一个然后才能使用，比如，当jifa了北冥神功，你就可以使用北冥的吸内力，如果需要太极神功配合，那么就要jifa太极神功。
对于武器空手的功夫，你如果不激发，那就是用 基本功夫。

```
比如：
你没有激发特殊功夫英雄剑法（jifa sword hero-jian;）当前的状态时无none（jifa sword none)。
这个时候你拿起一把剑，战斗中使用的是基本剑法，命中和伤害等按照你基本剑法等级计算。
当你激发以后，你激发学会的英雄剑法以后，战斗中会出现使用特殊功夫的描述。
```

```
jifa;
```

这个命令是列出当前的jifa情况，基本武功对应的特殊武功链接情况，还有个 “有效等级” 参数；

有效等级 = 基本武功等级/2 + 特殊武功等级

这就是为什么，jifa特别重要，特殊武功特别重要的原因，基本武功其实是50%的战斗效力。 powerup也是通过增加有效等提升战斗力，特殊技能special might 也是根据有效等级和加力的数据提升有效等级的。

备技能，互备，互搏

注意：先jifa好特殊功夫，再备。如果要换，要按顺序bei none;或者wbei none;之后，再jifa其它技能，换好了以后再bei,wbei 之后才完成。

互备，指同时激发两种武功在身。战斗时左右手分别使出不同的武功。分为空手互备(bei sk1 sk2)和武器互备(wbei sk1 sk2)。其中武器互备需要左右手装备相应兵器。互备增加0.5攻速，但是会降低气势获取速度，是取得互搏之前，增加攻击速度的一个选择。

空手备bei

这个命令是指定空手武功用哪一个。如果不指定的话呢，默认人物会使用 基本拳脚unarmed。
你可以选择如bei strike; 来使用掌法。
也可以选择如bei strike hand; 备 掌法和手法。 比如：灵鹫宫的六阳掌和折梅手 bei strike hand; 当你把两个特殊功夫都已经激发以后。
备了什么技能，战斗中就是用什么技能 bei 的功夫如果 jifa了特殊功夫，如jifa strike liuyang-zhang; 那么，bei strike后，会自动使用六阳掌，而不是基本掌法。
可以用bei none 取消。
如果备了两种功夫，那么在战斗中一个回合，可以发两下普通攻击。

武器备wbei

wbei的概念和bei 类似， 只不过是这个特指 武器武功，如武当派的太极刀，太极剑。
武器功夫不需要wbei一个功夫，武器功夫是按照手拿的武器决定使用什么功夫，需要和jifa配合。
比如单手拿剑，那就是用jifa sword 功夫名， 的剑法功夫， 如果没有jifa， 使用基本剑法， 如果没学过，就相当于基本剑法=0开始使用，战斗中会随机学到一点点基本剑法。

wbei 两个武功之后，双手各持一样武器，如blade，changjian，就可以使用刀法和剑法，也是一个回合使用两个功夫。

使用特殊功夫也需要jifa以后才能用。

左右互搏

左右互搏是和周伯通学习画方圆，习得的武功，不出现在武功sk列表里。
作用是每一回合多打出一次武功。一回合里 一招变两招， 两招变四招（和bei，wbei配合）
这里的一招指的是武功平砍的次数。

打架指令

比武fight

你和大部分人比武，点到为止，可以用fight比较安全，打到半血就可以就会“虚晃一招，跳出圈外”，不打了。 个别NPC不跟你来这一套：他可能会拒绝比武；还有npc直接会叫杀:如官兵，锦衣卫，大内的侍卫。不要跟官府的NPC动手动脚的挑衅。如果要你战胜NPC可以用这个命令。或者当你不确定是不是打得过这个npc的时候可以尝试一下，你输了他就停手了。命令就是fight 名字;

袭击hit

这个命令是在人家不和你fight时候，可以用。一般的打胜了，或者打晕了，就停手了，不会要NPC命，npc原则上也不会要你命。但是有的NPC在你反复挑衅打人家的时候会叫杀，跟你玩命，不死不停手。

这里还有个hitall命令，表示对在场的所有人发出hit命令。

```
hitall
```

这个命令也可以附加名字，表示对某一类人，hitall。 如：在对付多个劫匪的时候：

```
hitall robber
```

杀kill

就是跟人家玩命了。
命令就是:

```
kill 名字
```

这里也有killall 方法和hitall 一样，就是对所有人叫杀。killall 名字 也可以对某一类人叫杀。

killall在襄阳很有用，如果你在襄阳城的卫兵面前用kill命令，不管对谁下杀手，襄阳士兵们都认为你是对襄阳不友好，残骸百姓，对襄阳城有危险，所以都会对你进行叫杀。做慕容复任务杀叛徒时，还要避免被叫杀，可以用killall pantu;来解决。

叫杀

叫杀的意思就是，NPC会杀你。一般不会他们就直接开始战斗，如果“叫杀”就是有红色字提示你：XXX想杀死你。老玩家们称作 叫杀。

有一些技能会引起叫杀在本来不叫杀的战斗中。比如，空手的busy功夫，如天龙一阳指的fuxue，北冥神功的吸内力。即使你是fight 对手的，当你使用了这一类技能，不管成功失败，都会叫杀。你还是fight对手，对手会kill你，就是说你赢了会停手，他赢了输了都不会停手。还有一些技能是一半是不叫杀，另一半叫杀，就是一个技能两次攻击。还有技能是不成功不叫杀。

杀朝廷的人会被通缉，当你再次碰到朝廷的人，他们会主动叫杀，如：

-  在北京杀官兵，侍卫，锦衣卫。康亲王府的侍卫会在北京的地图范围内追杀。大部分通缉会过一段时间停止通缉，有文字提示你，可以给贿赂扬州的某位官员疏通一下，取消通缉。

-  在襄阳被通缉，襄阳官兵会原地叫杀，他们不会追你。注意，郭靖也不会放过你的，别去见郭靖。襄阳城的叫杀必须下线再上才行。

busy

就是英文的“忙”，意思就是你很忙，不能做你想做的指令了，就相当于“锤晕”等效果，没受伤害，但是不能做接收任何指令了，就呆在那里。

北侠的战斗模式是回合制，你一次，我一次。被busy了，就是几个回合不会动，不攻击，也无法回复，防御躲闪也大幅下降。所以，北侠的战斗中会busy的功夫很重要，但不是每个门派都有这样的功夫。

发招perform

这个命令就是使用高级功夫中的招法，一般老玩家们使用“pfm”一词，指代 perform 。

你能发什么招，请用 verify 命令，也请查看 wiki 中各门派武功的介绍。

[目标]可加可不加，不在战斗的时候如果想偷袭，需要加上目标；如果当前已经有一个战斗目标了，直接可以发招；有多个目标可以指定。

新格式：

```
perform 高级功夫.招法 [目标]
如：
perform liumai-shenjian.liumai [pantu]
```

使用新格式发招时，大多数的高级功夫无需激发（enable/jifa）到相应的基本功夫上，也无需准备（prepare/bei），可直接使用。少数高级功夫仍然需要激发到相应的基本功夫上。比如玄铁剑法必须激发到基本剑法和基本招架上，才能使用“海潮汹涌”这个招式。

旧格式：

```
perform 基本功夫.招法 [目标]
如：
perform finger.liumai [pantu]
```

使用旧格式发招时，必须把高级功夫激发（enable/jifa）到基本功夫上，才能运用高级功夫中的招法。如果是空手功夫，还必须准备好（prepare/bei）。

运功exert/yun

运yun命令也可以是exert ， 意思就是运用特殊内功的招法，如：

```
yun recover;     yun regenerate;    yun inspire;     yun heal;     yun lifeheal;     yun powerup;   yun shield
```

这个是常见的几个yun的命令例子，你所用的内功不一定都有，也不一定就可以用（有条件的）。简单解释一下：

****
