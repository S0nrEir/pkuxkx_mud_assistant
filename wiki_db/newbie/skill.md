技能学习

技能可以通过读书、练习、学习、切磋（分有效和无效，有效切磋的条件为：skill_level < EXP^(1/3)）、修炼、领悟等方式来提高。

利用潜能效率依次为：读书 = 练习 = 领悟（不消耗潜能） > 学习（正常消耗潜能） > 有效切磋（五倍消耗潜能） > 修炼（十倍消耗潜能） > > 无效切磋（1潜能=1技能小点）。

文化技能上限不受经验限制，其它技能上限与经验的关系为：skill_level = (EXP*10)^(1/3)+1（30悟性时）。

读书

可以读的书参考书籍一览。

不消耗潜能。

有的书用du 有的用study 有的用read，有的通用。

```
read tianlong 1
du shediao 1
du book for 1
```

练习

大部分特殊武功可以通过 lian 的方法自己练习，**条件为：基本武功>特殊武功**。

需要激发武功。练习空手武功时，要空手，练习兵器武功时，要手持兵器。

不消耗潜能。

```
lian strike 1
lian sword 1
```

学习

向师傅或特殊npc学习技能。

正常消耗潜能。

```
xue 人 for 技能 1
```

切磋

学习超过了师傅上限，自动变为与师傅切磋。

有效时五倍消耗潜能，无效时1潜能=1技能小点。

在只能通过切磋提高技能时，再考虑切磋（**尽量避免无效切磋，潜能消耗巨大**）。

比如“辟邪剑法”，东方不败能教到490级，但是练习条件是500级，所以只能切磋10级。

修炼

特殊内功100级以后，可以修炼提升。

十倍消耗潜能。

```
xiulian taiyi-shengong
```

只能修炼公共内功、本门内功、和出师前门派内功。
经验大于200M才能修炼出师前门派内功，上限为800级。经验每提升50M，提高修炼等级上限100级，提高修炼潜能消耗30%。
也就是，250M的时候可以修炼到900级，潜能消耗是正常的130%；300M的时候可以修炼到1000级，潜能消耗是正常的160%。

领悟

在通过少林的“罗汉大阵”后（俗称过阵），可以自己lingwu基本武功，**条件为：特殊武功>=基本武功**。

不消耗潜能。

正确jifa/enable 相应的功夫以后

```
lingwu sword 1
```
