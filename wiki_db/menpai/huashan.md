华山派

五岳剑派共由江湖中的五大门派组成，既：华山派，恒山派，嵩山派，衡山派及泰山派。势力庞大，隐然可与少林武当分庭抗礼。华山派原为五岳剑派中势力最强的一派，但华山两位前辈岳肃和蔡子峰因时机巧合，得以一窥当世无上武学密籍「葵花宝典」；但却又因见解不同，一人主张以修气为本，教导弟子重修气，另一人却以为因当以剑法为主而教导弟子重剑法，华山派至此竟分裂为剑、气二宗。二宗各持己见，终于水火不容，同门操戈！高手伤亡怠尽，华山因此式微。

**注意：本门派定位是纯门派，从本门派出师到其他门派或从其他门派出师到本门派，本门派的技能效果会受影响，请慎重考虑后再加入！**

门派摘要

华山派是北侠世界创世门派之一，许多传奇大佬第一个号都选择了华山。经过历史的沉淀，华山作为北侠十二大门派之一，其弟子在各方面均有不俗表现，有稳如老狗血厚内力高的气宗，有剑走偏锋攻高血短的剑宗，加之多年优化的技能体系，可谓北侠六边形门派。华山派弟子基本可以胜任北侠所有任务，没有明显短板。

**华山分剑气二宗，华山派弟子可在派内自由切换（有CD）剑气二宗归属。**

**剑宗自视甚高，不接受别派出师而来，带艺投入华山之人。而气宗则较为开明，无此限制。**

**天赋选择：**

华山各方面较为均衡，天赋选择较为灵活，没有特别突出的Biuld方案。很难说哪种更好。天赋对应职业突出就好，白虎伤害更高对无群攻的九剑平砍更好，但是前期的生存能力弱，朱雀命中攻速更好，前期更巴适，玄武不怕pk任务更安逸，青龙中规中矩哪都不弱哪都不强。

**职业匹配：**

华山弟子选择四大职业均有不错的匹配度，根据先天属性选择相对应的职业即可。

若有悟性特长者可考虑成为一流青龙，对有藏经阁破绽的玩家，效果强化。独孤九剑普攻+10%伤害。但是依然与白虎伤害相差甚远。

华山门规

-  华山弟子拜师后，复又叛师超过十次者，永不能重归华山。华山一概不收。

-  **特别注意：华山弟子若为非作歹，滥杀无辜而导至道德少于-100者，将被逐出师门，所有武功减半！ **

-  东厂人物，即辟邪剑法超过50等级者一律不收。

华山内功(huashan-neigong)

-  基本功效：可激发为force。

-  学习条件：基本内功大于10级。

-  练习条件：达到师傅的最大级别后靠修炼和切磋增加等级。

-  特殊功效：

-  自疗 [exert/yun heal]

-  功能：回复当前气血。

-  要求：华山内功20级，当前内力50。

-  加力 [exert/yun powerup]

-  功能：暂时增加攻击命中和有效轻功。

-  要求：紫霞神功和华山内都大于600级时，达到最高系数13%，比其它powerup少2%，非剑宗弟子只能获得最差效果（10%）。

-  激发效果：永久增加剑法有效等级和剑法伤害，受华山内功级别、先天身法、基本剑法级别影响，内力上限计算从有效内功*10变成有效内功*8。

-  加力特效：对于非剑宗弟子，加力内力消耗为80%，剑宗弟子加力内力消耗在80%基础上，每500级减少15%，最高2500级时，减少75%。

-  限制：非剑宗弟子最高只能修炼/学习华山内功到1000级,已学习的可以使用，但无*剑宗被动*加成，内力上限加成有效。

-  防御 [exert/yun shield]

-  功能：吸收物理伤害

-  效果：增加伤害抗性8%，内力消耗6点/秒。1000级后，效果和消耗翻倍。

-  剑宗被动：增加10%的剑法[jifa/enable]效果和更低一些的伤害。这是华山剑宗的核心。

紫霞神功(zixia-shengong)

-  基本功效：可激发为force。

-  学习条件：**先要问岳不群回归气宗**。基本内功大于10级。紫霞秘籍能读到101，岳不群有150级紫霞学习必须是气宗，高矮长老有200级紫霞，200级后自己修炼

-  练习条件：达到师傅的最大级别后靠修炼(xiulian)和与师父切磋(qiecuo)增加等级。

-  特殊功效：

-  自疗 [exert/yun heal]

-  功能：回复当前气血。

-  要求：紫霞功20级，当前内力50。

-  加力 [exert/yun powerup]

-  功能：暂时增加命中和闪躲百分比，对于气宗弟子将会得到极大增强，紫霞达到1500时，达到北侠最高系数18%。非气宗弟子只能获得最差效果。

-  最低要求：紫霞功30级，当前内力100。

-  激发效果：永久增加精神和气血，受紫霞神功级别、华山内功级别、先天悟性影响，内力上限计算从有效内功*10变成有效内功*12，内力恢复速度提升。

-  限制：非气宗弟子无法修炼/学习紫霞神功,已学习的可以使用，但无*气宗被动*加成，内力上限加成有效。

-  防御 [exert/yun shield]

- 功能：吸收物理伤害

- 效果：增加伤害抗性6%，内力消耗3点/秒。1000级后效果和消耗翻倍。

-  气宗被动：增加气血和精神若干，数量取决于先天悟性。紫霞增加的精神比少林少，气血比武当少，但是二者兼得。(这个必须在气宗才加气血精神，出师带不走，最低条件是紫霞神功华山内功大于60）

华山派新手任务

作为北侠江湖十二大门派之一，华山派为新入门弟子提供了本门独有的新手任务，本派弟子不需要参与各类钓鱼、唱戏、配药等等的公共新手任务，本门新手任务可以完全提供从离开新手村直到初入江湖（20w经验）所需的经验。

华山巡山任务

新人拜师华山以后可以问岳灵珊领取新手巡山任务

```
ask yue about job
```

岳灵珊给你一块令牌，持有令牌期间，走过华山地图所有房间，除去岳不群寝室、音乐练习室、思过崖后山部分、紫气台部分。

每到一个房间会得到“你在xxx巡弋，尚未发现敌踪。”的提示，即可前往下一房间。完成所有房间巡弋后，将令牌交给岳灵珊，就算完成一次任务。

```
give ling pai to yue
```

另：如令牌丢失等情况需要放弃任务，找岳灵珊输入：

```
ask yue about fail
```

放弃任务无惩罚，但需要一定cd才能重新接取任务
**Tip:**

-  **下线或者完成时间过长会导致令牌丢失，从而会导致任务失败。**

-  这个任务可以用来学习怎样制作触发（trigger）和别名（alias）,特别是在途中有“痴汉”拦路的情况下怎么解决，对新人掌握基本的机器人制作颇有好处。

华山指点弟子任务

巡山任务获得一定经验后，岳灵珊不再发放任务，会告诉你去找她老娘，也就是掌门夫人宁中则领取下一步任务，找宁中则领取“指点弟子”任务。

```
ask ning about job
```

宁中则告诉你去练功房指点一下弟子。去到练功房，看到有 xxx教导的华山弟子（xxx's student）,开始指点

```
zhidian xxx's student
```

指点几次以后，弟子会开小差跑掉

```
xxx趁你不注意，一溜烟不知道跑到哪里去了。
```

此时需要满山去找跑掉的弟子，找到弟子后，让弟子跟你回去

```
ask xxx's student about 指教
```

多问几次以后弟子同意跟你回去，弟子开始自动跟随你，直接回到练功房，继续指点。

多次指点后，获得提示

```
xxx一招一式有板有眼，你可以回去和宁中则复命了。
```

这时可以去找宁中则交任务了。

```
ask ning about finish
```

另：如弟子丢失等情况需要放弃任务，找宁中则输入：

```
ask ning about fail
```

放弃任务无惩罚，但需要一定cd才能重新接取任务

**Ttip:**

-  比较麻烦的是弟子跑了以后需要满山去找，聪慧的同学这时就可以学习做路径自动跑路了。

向玩家问好任务

华山送信任务

指点弟子获得一定经验以后，宁中则会告诉你，让你去找掌门领取下一步任务，去找宁中则身旁的华山掌门岳不群领取“华山送信”任务。
**该任务与主流任务“华山送信任务”所有完成流程完全一样，区别仅仅是抢信盗贼的难度不同以及所获得的奖励不同。**

接任务前的准备

**华山派新手请先设置变量，输入：**

```
set newbiejob 1
```

设置该变量后，找岳不群接取的就是华山新手送信的任务，否则就是所有玩家都可接取的主流“华山送信任务”，难度较大，抢信贼会秒了新手。

除了设置变量，在接取送信任务前，请先full自己的各项技能。因为送信任务是需要战斗的，有一定危险。

在完成了“巡山”和“指点弟子”的任务后，积累了一定的经验，可以支持一定等级的武功技能，这时候通过找师傅学习（花费潜能）或者阅读各类书籍书籍一览将自身各项技能练至最大值，俗称full技能。

并通过打坐（dazuo）、吐纳（tuna）将自身精神和气血提高一定程度。

新手们可以去领取免费的装备

```
扬州中心广场 -
down
out
发呆室的爱爱为1m以下新手提供免费装备，具体命令为list(查看装备列表)，draw装备id以获取装备,下线(quit)前有效，可以每次上线先去领一身装备。
wear all 或者 wear xxx穿上各种防具或衣服
remove all 或者 remove xxx 脱衣服或防具
wield all 或者 wield xxx 拿起武器
unwield all 或者 unwield xxx 放下武器
```

做好装备、技能、属性的准备后，就可以去接送信任务了。

任务流程

找岳不群接任务

```
ask yue about job
```

岳不群给你一封密函，要你送给在某地的某人，地点可能会以图片形式显示。
在去往指定地点期间，会在你身旁跳出来两个抢信的盗贼，即刻进入战斗，盗贼比较凶猛，注意随时逃走，逃走后，盗贼不会移动，会在原地停留一小段时间，尽快恢复好，再次进入战斗。

杀死盗贼后，从尸体上获得战利品，以及把抢走的信再拿回来。

```
get xxx from corpse 或者 get all from corpse
```

拿回信后就可以淡定的去找送信目标了。

Tips:

-  如果被盗贼打死打晕如果能可快速返回现场还有再次战斗的机会，但如果回到现场后盗贼消失，则任务失败，回去找岳不群ask fail吧。

找到送信目标后

```
songxin xxx
```

出现黄色字体对话，则任务完成，回去找岳不群复命。
回到岳不群

```
ask yue about finish
```

本次任务完成，获得奖励

另：如实在找不到对应送信目标，或者密函丢失等情况需要放弃任务，找岳不群输入：

```
ask yue about fail
```

放弃任务扣除少量经验值和潜能，再次接取需要等待较长CD。

**Tips:**

-  被盗贼打死，或者因逃跑等原因造成盗贼消失，则密函无法找回，任务失败。

-  密函在身上时，主动退出下线（quit）则密函丢失，任务失败。

-  掉线，或者直接关闭客户端不属于正常下线，人物和身上物品会保持一定时间，直到重新连接，任务仍然可以继续。但掉线时间过长仍然会自动下线导致任务失败。

-  一些高危地带新手不要去送信，例如白驼山、灵鹫宫、苗疆等等，遇到此类地方直接放弃任务。

-  如果没等到盗贼来抢信，直接将信送给目标，也可以完成任务，但是岳不群给的奖励。。呵呵，比较感人~~
