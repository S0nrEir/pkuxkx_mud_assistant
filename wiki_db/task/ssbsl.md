杀手帮试炼

---

- 任务背景：这是一款洗衣机制作的生存小游戏，要躲过里面各种NPC或者玩家的追杀并反杀活到最后。

- 任务规则：满8人开始，初始9个地址,3分钟一次,少两个地图,直到最终只剩下一个地图,(每个人身上就8个格子);战斗独立,每人进来都是100的血,不会自动回血;每次行动加1经验,战斗多(3)倍经验,击杀经验最高,升级加属性;回血靠捡到的道具，每次行动后有CD(5秒)(CD内多次尝试有惩罚)，最后一个活下来的人获胜

- 任务地图：初始9地图,直接go 编号过去,不需要走地图格子,依次为:“河边”, “森林”, “山道”, “民居”, “铁匠铺”, “药房”, “药田”, “练武场”, “洞穴”

- 任务要求：无，只要您会敲键盘，会唱打不到我吧没有办法我就是这么强大，追不上我吧啦啦啦啦啦啦~~~~~~~

- 任务模式：假死

- 任务CD：游戏每天可以无数次参数

- 任务掉落：无平时装备宝石等掉落

- 任务奖励：第一名有30万经验,第二名15万,第三名10万,公式为:30万/名次，每天一次奖励

- 经验加成：无，本任务为固定经验模式

- 任务流程：杀手帮山顶找试炼管理者无心ask wu xin about 报名，进来后来到场内山顶待机悬崖处，这时候如果有六名及以上玩家就可以不需要输入ready直接倒计时开始，如果人数不足六人则需要一起都输入ready才可以开始，每次上限八人名额，如果玩家不足便由电脑AI NPC代替

这里我一个人参加就要对付另外7名NPC，NPC的名字都是些大佬，好怕怕赶紧go 1-9逃跑~~~~

小游戏里的人物属性跟外界平时练级的游戏属性是无关联的，大家都一样，sc可以看到人物状态，可以通过find找东西来一点点升级,但是要注意你升级的同时NPC也会升级所以不要一直呆在一个地方作死升级

战斗是回合制的有指令固定cd,你一拳来我一脚看谁运气好能闪躲就占先机，气血到0以下就出局了，打不过可以中途go地图数字逃跑但是要注意气血和指令cd,指令没到cd持续输入是会有cd惩罚的。

这个游戏不能一直呆在一个场景，因为地图会一定时间随机少两个地图，有可能你在的这个地图会出现毒雾，快出毒雾时系统会有个30秒的倒计时提醒，最后五秒会一直提醒，如果倒计时结束你还在这个房间就会被毒死直接出局game over了~ ><,

把所有人淘汰后你就赢了，看我多厉害把传说中的五绝大佬都淘汰了苟到了最后哇哈哈~~~~出来后会自动获得奖励，一天就一次获得奖励的机会。

下面介绍下基本指令与物品道具

| 指令 | 指令用途 |
|---|---|
| ready | 游戏开始前准备 |
| wait | 取消游戏准备 |
| tuichu | 游戏开始前退出游戏 |
| go | 输入go 加1-9数字移动，一共9个可移动场景 |
| map | 查看当前能移动的地图 |
| kan/see | kan/see加物品id查看指定物品 |
| sc/score | 查看当前角色属性(试炼专属属性) |
| b/bag | 查看当前角色背包(试炼专属背包) |
| up | up加装备id：升级指定装备 |
| hecheng | hecheng加物品id：暂时只支持合成钢,也就是hecheng gang |
| use/y | use/y加物品id:使用道具,暂时只支持4种回复道具 |
| find | 核心动作之一,查找当前地图内的物品,可以加经验升级人物等级哦 |
| k/kill/h/hit | k/kill/h/hit加人物id:攻击指定人物 |
| game | game加要说的话:此生存游戏局内聊天 |

所有图能都掉任何物品,但是单独的地图有掉落加成(后续优化)，大概率指超过或等于10%，小概率指不到10%都是用find来寻找

| 场景 | 掉落物品 |
|---|---|
| 河边 | 大概率: 鱼肉, 火石, 石头 |
| 森林 | 大概率: 木头, 兽皮，小概率: 草药, 铁矿 |
| 山道 | 大概率: 石头, 人参，小概率: 铜矿, 铁矿, 木头, 兽皮 |
| 民居 | 大概率: 火石, 兽皮，小概率: 金创药 |
| 铁匠铺 | 大概率: 铁矿, 铜矿，小概率: 钢, 火石 |
| 药房 | 大概率: 金创药, 人参, 草药 |
| 药田 | 大概率: 人参, 草药，小概率: 木头 |
| 练武场 | 大概率: 兽皮，小概率: 木头, 鱼肉, 金创药 |
| 山洞 | 大概率: 铜矿, 铁矿, 石头, 兽皮 |

恢复道具有：草药(20), 鱼肉(30), 人参(50), 金创药(80)，括号里是恢复数值，

合成素材有：木材, 石头, 铜矿, 铁矿, 水, 皮, 火石

装备提升: 命令:up，按木→石→铜→铁→钢的顺序提升，皮为一直需要的材料

材料合成: 命令:he，1铁+1木+1火石 ⇒ 2钢

合成列表

| 合成材料 | 合成成品 |
|---|---|
| 三份药草 + 一份人参 | 一份金创药 |
| 一份铁矿 + 两份铜矿 | 一件陷阱 |
| 一份鱼肉 + 一份木头 + 一份石头 | 一份鱼烩 |
| 一份兽肉 + 一份木头 + 一份火石 | 一份烤肉 |
| 一份铁矿 + 一份铜矿 + 一份火石 | 两份钢 |
