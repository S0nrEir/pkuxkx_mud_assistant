8.小窗
如何用小窗来显示fullme图片

```
1.你需要先触发一下，比如输入
#trigger {^???http://pkuxkx(*)$} {#mxp http://pkuxkx%1} fullme 0 1 0 fullme

2.建立一个小窗,其中内容为
::MXP
```

小窗可能用来显示聊天信息

```
1.假设聊天信息你都#cap chat了

2.小窗内容设置为
::link chat
```
