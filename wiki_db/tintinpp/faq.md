实战篇

#map get 怎么用？

参考：2)

-  map list检索时，需要添加variable标识后面是结果变量，像
```
#map list {} {} {$desc} {} {} {variable} {mapResult}
```

;

-  map get 是不用variable标识的，直接
```
#map get roomdesc {desc}
```

desc就能取到roomdesc的值了。

#action 的优先级如何设定？

参考：3)

#action 的默认优先级是5，数字越小优先级越高。最高优先级为0。

为什么触发器会失效？

参考：4)

**问：**做了一个简单的打坐机器人。发现有的时候会突然失效，对触发语句没有响应。但是钓鱼的时候，就没有这样的问题。请问，这个大致是什么原因造成的呢？

**答：**先确认是不是网络慢，丢包引起的。有时传过来的文字因为丢包只有半句或者乱码，造成 action 不匹配而停止了。你看看是你的网络ping 服务器相应时间是多少？如果比较长，建议设定

```
#CONFIG {PACKET PATCH} {0.5}
```

这个 0.5 应该设定多大的值还是多少，具体看你的网络相应时间。

如果网络状况长期处于不佳状态，也可以将该设定语句直接写入启动配置文件，如：WinTin++ 的 main.tin 文件。
