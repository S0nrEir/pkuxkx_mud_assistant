解密：晋阳药铺

药浴是新玩家可以完成的一项简单解密任务，完成后可以获得1-3点的先天根骨。

新手在积累了一点金钱以后（通过例如抄经等任务），可以来到**扬州当铺**购买所需要的药材来为药浴做准备。

药浴所需物品清单

-  **大木桶**（问郝巴要，若已经给过蛇床子，要不到的话需要quit后再要，最好是**未用过新的木桶**）

-  **生甘草**（但是生甘草贵得多，可以买便宜的甘草bo gancao得到）

-  **地骨皮**（地骨皮贵得多，可以买便宜的枸杞xiao gouqide得到）

-  **蛇床子**

-  **温泉水**（可以用水壶等容器装）

攻略

**以下流程供新人参考。**

来到**当铺**后分别输入：

```
list 蛇床子
list 生甘草 (或甘草 bo gancao)
list 地骨皮 (或枸杞 xiao gouqi)
list pot   (后续热水用到,可一并购买)
```

得到**商品序号**后输入：

```
buy 对应商品序号
```

**初次泡澡：**

购买2个蛇床子，1个生甘草或甘草，1个地骨皮或枸杞

**已经给过郝巴蛇床子：**

购买1个蛇床子，1个生甘草或甘草，1个地骨皮或枸杞

**重要提示：**当铺里的商品是玩家卖进当铺的，所以有**数量限制**，若是当铺断货了，可以参见补充提示来获取洗澡所需的药材。

接下来前往晋阳的药铺，晋阳从扬州出发需要过黄河。若你还可以使用路引，可以travel jy到达晋阳。

**小提示：**建议先充值行旅通。过河可以用node walk yz_qfdk1 来到黄河的曲阜渡口，ask 渡口的npc about 过河，enter指令进入筏子，到对岸后即可通过node或walk指令走到晋阳。

从**晋阳的广场**出发到**晋阳的药铺**路线：

```
sw;w;w;w;s
```

来到药铺，郝巴进行对话，**第一次**拿大木桶:

```
ask hao about 帮助
give hao zi
ask hao about 交个朋友
ask hao about 独门秘方
ask hao about 木桶
```

成为郝巴的朋友后以后再拿大木桶（刚刚拿过重新再拿需要quit）：

```
ask hao about 木桶
```

随后回到扬州，然后去**扬州杂货铺**购买一个水壶(Pot):

```
buy pot
```

然后前往信阳温泉，从**扬州中心广场**出发：

```
do 4 w;nw;w;w;se;sd
```

到达**汤泉池**以后：

```
fill pot
```

回到扬州，前往扬州的**贵妃浴场**，贵妃浴场在盛昌总行西边，

花费30两银子进入澡堂(give zhaodai 30 silver)，若提示“咦？你不是刚进去过么？”可以再次输入enter强行进入：

```
look mu tong —-查看木桶编号
add zi to <木桶编号> --加入蛇床子
add pi to <木桶编号> --加入地骨皮，如果你买的是枸杞需要先xiao gouqi
add sheng gancao to <木桶编号> --加入生甘草，如果你买的是甘草需要先bo gancao
do 3 pour tong --将水壶里的热水倒入木桶，注意身上最好不要带其他液体容器
xizao <木桶编号>
```

把**三样药材**置入大木桶,再倒入**三次以上温泉水**后，就可以跳进大木桶开始洗澡，经过**一定的时间**以后，大木桶裂开（有极小概率不裂，无影响，丢弃木桶即可，重复使用同一个木桶可能会遇到bug），洗澡完成。

这时可以输入note -a来看看自己是否洗澡成功了，若成功则会有如下系统笔记：

```
XX岁时经历药浴。（XX为最近一次洗澡的岁数，多次洗澡不会重复显示）
```

增长岁数以后药浴只需要买好药材，装好温泉水，拿到大木桶，就可以再次在浴场洗澡。

从14岁开始，**每长一岁可以洗一次**，完成药浴**四**次以后（最早在17岁完成）会有如下系统笔记：

```
你在幼年持续药浴浸泡中，获得了1点根骨。
```

补充提示

-  除了大木桶必须找郝巴领取外，其他三味药材一般能在扬州当铺买到

-  蛇床子是在襄阳药店、扬州药店、苏州药店均有可能出售

-  甘草在白驼戈壁、明教沙漠边缘、赞普边上的天溪处有，zhai gancao 然后 bo gancao 得到生甘草

-  灵州酒馆有卖枸杞，购买后xiao gouqi 得到地骨皮

-  枸杞、甘草需要加工后才能使用

-  泡澡得用热水，用水壶装温泉水

-  现在条件放宽到24岁，在24岁之前泡澡4次会增加先天根骨

-  现在可以在扬州贵妃浴场中洗澡，不必购买储藏柜（2023年）

-  信阳温泉的位置从扬州出西门可以找到

根据hash大侠的提示：

```
灵州酒馆买枸杞然后xiao gouqi 得到地骨皮
甘草在白驼戈壁、明教沙漠边缘、赞普边上的天溪处有。zhai gancao 然后bo gancao 得到生甘草
扬州当铺买蛇床子
去扬州杂货铺买水壶(Pot)去信阳温泉灌热水 灌热水的命令是 fill pot
找郝吧要木桶（第一次需要上面和郝巴交涉的过程，以后直接ask hao ba about 木桶就可以了）
把所有东西灌木桶里回家泡澡（家在扬州牙人那里买，100gold，18岁可买，买了之后牙人那里enter 你自己的id 就可以进家门了）
18岁可以泡到24岁，而且18岁之后不fullme得话年龄是不会增长的
一直处于fullme状态下的话，24小时在线时间年龄增加一岁
```

提示：

-  蛇床子是在襄阳药店、扬州药店、苏州药店均有可能，当然更直接在当铺购买更方便。如果都买不到，现在在杭州玩家当铺有售。

-  只要连续洗澡4次即可获得先天，最快14、15、16、17四年，在17岁时完成。

```
经过少年时期药水的不断浸泡，你觉得自己的先天根骨有了明显提高！
```

以上是先天根骨提高的提示。

泡澡命令

```
fill pot 温泉灌水用
-----然后去扬州贵妃浴场，花30silver，进入澡堂子
-----注意：下面是洗澡要加的三样料。别少加了。不然白洗，浪费一年。。
look mu tong （查看编号）
add zi to <木桶编号>
add pi to <木桶编号>
add gancao to <木桶编号>
do 3 pour tong
xizao <木桶编号>
```

补充：少加一味药是可以bath的，所以bath以前一定要检查一下，因为每岁只能bath一次，以免白白浪费呀。
