北侠短视频

---

- 短视频发展史：2013年8月由新浪领投的一下科技推出现象级产品——秒拍。借助微博的独家支持以及众多明星资源的入驻，其用户量迅速达到千万级。短视频至此开始进入大众视野。14年5月，美图秀秀推出革命性产品——美拍。原创性的人像特效、视频特效技术，在广大女性中好评如潮。15年5月小咖秀上线，其独特的对嘴表演模式吸引了大量用户参与和分享。此时，短视频拥有了第一批用户，开始在人群中传播。但由于缺少优质内容制作者、缺乏清晰的赢利模式、难度过高的大众参与门槛，短视频还未形成明确的产品定位。16年9月抖音加入战局，独特的美颜效果、便捷的音乐短视频制作以及针对年轻人的定位，吸引了95和00后的大量关注。同年梨视频、papi酱等短视频创业者等的崛起，让投资者认清了短视频领域的投资潜力。17年1月抖音获字节跳动天使轮融资，张一鸣豪掷重金强势扶持抖音。17年3月，岳云鹏转发了一条其模仿者的短视频，视频下方印有抖音字样——引爆了上线一年多的抖音。同年9月，抖音开始与爱彼迎、雪佛兰、哈尔滨啤酒展开视频广告合作，开启商业化运作之路。2022年4月20日Z神的北侠版短视频功能上线，从此大佬们的战斗风采无处遁形。

- 短视频制作：short_video start 加名称开始录制短视频，short_video stop停止录制短视频

short_video list 查看列表，short_video watch 观看已录制

视频太长的话会分页显示

- 系统随机短视频：这项功能让大佬们的战斗风采无处遁形，系统每天会自动抽取一个玩家录制一个最佳战斗短视频并公布出来，萌新们终于有机会知道大佬们战斗时是多么的厉害，大佬们瑟瑟发抖~~~~最近段时间某大佬的某暗器爆炸伤害就被萌新们议论纷纷求教。入选了每日最佳战斗短视频的玩家可以在洛阳关帝庙找北丑领取今日最佳战斗津贴，如果没有入选而去问这个问题会被关小黑屋罚站的哦，津贴是啥内容俺也不知道自己探索，另外如果多次入选每日最佳战斗短视频还有个稀有title奖励，没什么实际用处但很稀有，也不知道是啥写的时候还没有人获得。

用short_video published指令可以查看所有发布的视频，系统视频发布者就是system，用sv play <视频ID>播放。

- 短视频相关指令：

| 视频指令 | 指令用途 |
|---|---|
| short_video list | 查询自己录制的所有短视频 |
| short_video start <视频名称> | 开始录制短视频，如果已经存在将会覆盖 |
| short_video stop | 结束当前视频录制 |
| short_video watch <视频名称> | 观看录制的视频 |
| short_video share <视频名称> | 分享该视频 |
| short_video send <视频名称> to <玩家> | 分析该视频给具体玩家 |
| short_video published | 查询所有已发布的视频 |
| short_video published <查找条件> | 查询所有已发布的视频中符合条件视频查询条件可以是视频ID，发布人或者简介 |
| short_video play <视频> | 观看别人已经发布的视频 |
| short_video delete <视频> | 删除自己的视频，如果已经发布则删除发布信息 |
| short_video rename <旧名> <新名> | 给自己的视频改名 |
| short_video add_cover <视频> with <ASCII名称> and <标题> | 给视频加标题？ |
| short_video here or <物品> | 查看和当前环境或者某物品相关的所有视频 |
| short_video webaddress <视频> | 查询一个已发布的视频的网站地址 |
| sv watch/play <video> 后面加倍速，支持2、4、8倍速播放 | 短视频快进功能，格式举例 sv play yizhi 8x |
| sv reward < 视频> with <金> | 打赏功能，用gold打赏作者 |
| sv withdraw | 提现，平台收取30%手续费为以后玩家活动基金，打赏1金作者可以提现70两白银 |
