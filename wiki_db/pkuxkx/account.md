MUD帐号管理：

Mud帐号注册：

注册Mud帐号请先通过web版mud连线北大侠客行，输入new，按步骤提示注册Mud游戏帐号并设定密码。

注册过程中请注意：务必设置有效Email地址！这是在帐号密码忘记后找回帐号的唯一途径！

注册成功之后，在注册邮箱中会收到一封验证邮件，只要点击其中的验证链接，或将验证链接地址拷贝到浏览器中打开即可完成邮箱验证。

为防止骚扰，完成邮件验证之后，新手期内可以使用所有频道进行交流，否则只能使用QQ频道进行交流。

邮箱验证流程：

1、在注册时，如果输入的邮件地址有误，则有可能收不到验证邮件。这时也不必着急。在进入游戏后，可以再次输入register 电子邮件地址(例：register mymail@emailserver.com)，输入后请不要退出；

2、电子邮件地址需要是 xxx@xxx.xxx 的格式，如果确信地址格式正确但无法注册，可能是客户端把转义
了@字符，请关闭客户端转义功能或者从北侠web版 https://mud.pkuxkx.net/ 登录注册。;

3、登录邮箱收信，每个注册帐号会收到一封单独的邮件告知注册验证链接，如：

```
请点击 https://pkuxkx.net/xxxxxx 完成帐号邮件验证
```

按照说明进行点击即可完成注册

4、由于邮件服务器原因，某些帐号可能会长时间收不到验证邮件，这时可以用所注册
的电子邮件地址给register@pkuxkx.net发信，发信格式如下：

```
邮件题目：pkuxkx register
邮件内容：自己帐号的英文id1/英文id2/英文id3/英文id4
```

即：邮件内容是所有用此邮件进行注册的英文id，如果有多个，用“/”间隔开，中间不要带空格。

稍等一分钟左右，即可收到每个id的验证邮件，然后点击其中的验证链接 即可。

**注意：此种方式只有在注册mud账号时输入了正确的邮箱地址，或者register过正确的邮箱地址，但没有收到验证邮件的情况下才有效。**

5、如果要更换注册的邮件地址，同样可以register 新邮件地址，这时系统将先验证旧邮件
地址，再验证新邮件地址，验证流程如前面所述。

需注意：务必先在MUD里register 电子邮件地址，再进行下一步骤。发邮件时，如果邮箱
存在多个别名，注意选择所注册的邮件地址作为发件地址。2022年及之后创建的账号，如果忘记密码，只能通过register所确认的邮箱自助找回，如没有及时register及确认邮箱，巫师不再使用手动更改密码等方式帮助找回。

Mud帐号密码修改：

如果Mud帐号可以正常登录，可以在Mud中输入passwd命令，按说明修改密码。

Mud帐号密码找回：

如果Mud帐号密码遗忘或被修改，但之前用register命令注册过电子邮件，可以直接向register@pkuxkx.net地址发电子邮件申请密码找回（注意，register命令生效前注册的id无法用此方式找回，请跳转至下面“常见问题”的第一条）。找回流程如下：

1、如果记不起具体帐号，只记得注册邮件地址，请先按照以下格式向register@pkuxkx.net邮箱发送邮件：

-  邮件标题：pkuxkx id    （注意这里就是打id，不是打自己的id）

-  邮件正文：空

-  发件邮箱：必须是注册邮箱地址

系统将回复邮件，并告知用此发件邮箱地址所注册的所有帐号ID

2、找回密码，请按照以下格式向register@pkuxkx.net邮箱发送邮件：

-  邮件标题：pkuxkx password

-  邮件正文：自己帐号的英文id1/英文id2/英文id3/英文id4…（如果只改一个id，只写id1即可）

-  发件邮箱：必须是注册邮箱地址

系统将回复邮件，告知新密码和密码修改结果。请再收到邮件一分钟后用帐号和新密码登录，并根据系统提示修改密码。

3、如果发送邮件后长时间未收到回复邮件，请到论坛发贴或到主页查询官方QQ群，加入QQ群反映问题。

4、register@pkuxkx.net这个邮箱完全自动化处理，请不要在邮件中加入除了以上内容之外的任何内容，否则可能无法正确处理。

Mud帐号备份：

北侠Mud账号文件由系统自动备份，备份机制如下：

1、每个整点所有在线玩家档案统一备份一个版本，每个ID最多保留3个月内的24个较新版本，备份文件保存在Mud系统内。

2、每天零点过后所有玩家最新版本的档案打包备份，并传送到备份服务器上异地备份。异地备份保留一年内的每日备份，以及一年以上的每月第一天的备份。

3、2015年以前的数据做归档处理，请参考常见问题第4条

**系统备份只做档案出错或老玩家回归、自杀恢复时恢复使用，不能因玩家自身原因（如错误输入等）造成的损失而要求恢复备份。如怀疑档案有问题需要恢复备份，请直接到论坛bug投诉版按要求发帖，贴出符合要求的完整版log作为证据，不要私信或者在群聊里联系巫师。**

主页/论坛/wiki帐号管理：

主页/论坛/wiki帐号注册：

首先按照Mud帐号注册流程注册Mud帐号，并按照上面流程register，然后在Mud中输入命令genpin，

这个命令会向你已经验证成功的注册邮箱发送一个北侠网站注册/重置密码链接。

点击邮件中的链接后，可以设置北侠网站密码；如果网站账号尚不存在，将自动完成注册；

如果网站账号已存在，将自动重置密码。以后登录网页、论坛、wiki只需要账号和网站密码，不需要再输PIN。

主页/论坛/wiki帐号密码修改：

首先在Mud中输入命令register，确保自己已经register过，然后输入命令genpin，如果网站账号已存在，将自动重置密码。

可见Mud帐号密码是整个北大侠客行帐号密码体系的根本，再次提醒请务必设置有效的注册email地址，以防Mud帐号密码忘记或丢失。

常见问题：

```
1、 register命令生效之前注册的id如何找回？
```

目前北侠保存了超过20年玩家档案，但是其中某些档案由于经验过低（小于100）且长时间未登录，或者密码被盗，或者自杀备份等种种原因被封存，也有部分玩家长时间afk导致密码甚至id都忘了。这种id的找回请到QQ群联系icer，并提供档案相关的独一无二的信息，比如邮箱（虽然没有register验证，但也可作为证据，如果还能够使用就更好）、自建的alias、nick等等，icer可以帮助查找，并人工确认是否本人。

```
2、 注册、重置密码无效，论坛无法登录怎么办？
```

可能因为要注册的帐号之前已经有别人注册过，但是mud中帐号自杀清档了，而论坛帐号不可能同步自杀，当mud中此帐号重新建立时，会出现网站用户已经注册过的情况，这时只要重置密码即可。

论坛的绑定邮箱以及找回密码等功能都无效，因为北侠论坛账号只与mud账号绑定，有mud账号才能按照前面的密码修改方式修改论坛账号密码。但mud账号自杀不影响论坛账号，mud新注册的账号可以改同id论坛账号。总之：北侠所有账号都是mud账号，只认mud注册邮箱。

```
3、 论坛登录失败怎么办？
```

经常有玩家反应论坛无法登录，现将原因及解决办法说明如下：

**现象一：密码输入后屏幕中心出现提示：**

这种现象原因有二：一是确实密码错误，需要重新输入正确密码；而是这个帐号还没有在mud里register过，无法登录论坛，请参考前面的说明来register，再登录就可以了

**现象二：密码输入后没有任何错误提示，密码输入框又清空。或者验证码无法显示。**

这时请确认一下你输入的论坛地址是否为pkuxkx.net/forum，并在论坛首页随便找个帖子，看一下浏览器底部状态栏的显示地址是不是pkuxkx.net开头，或者在帖子上点右键，看属性也可以。如果如下图所示：

虽然网页地址是pkuxkx.net开头，但帖子的地址却以pkuxkx.com或者web.pkuxkx.net等开头，那么说明你的浏览器缓存了以前的页面，导致连接混乱，同样导致登录不成功。

请更换浏览器或者清空当前浏览器的缓存，具体清空方式与浏览器有关，请自行查询。
**注意，有的浏览器支持按住ctrl，再点刷新按钮，强制刷新当前页面，不读缓存。可以先试试这个方法**。

```
4、 之前的Mud帐号为什么登录显示不存在了？
```

北大侠客行自2002年之后的玩家档案都有保留。2015年北大侠客行服务器被黑，相关档案和代码被窃，具体请看江南侠客行来历。为了保证玩家档案安全，统一采取以下措施进行保护：

-  2015年2月之前已存在的ID，如果2015年2月之后再未登录，则统一封存。登录此类id时会提示档案不存在。如要恢复请联系巫师。

-  2015年2月之前已存在的ID，2015年2月之后存在大量ID相近时段同时登录的情况，则统一封禁。登录此类id时会提示密码被盗，请联系巫师。如要解封请联系巫师。

在此向受影响的玩家致歉，并再次谴责“静电清水（yhzzyahoo）”、“花错（huacuo）”以及江南侠客行相关巫师的丑恶行径！
