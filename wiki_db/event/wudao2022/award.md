**三人对抗赛**

*****************************

报名要求：

1. 限三人一组

2.自由组队，跟帖报名

3. 单人经验必须在 1000M 以下，无其他经验限制（999M 想跟 10M 组也可）。

赛制：

```
1. 小组将以抽签形式分配两两对战，一局定胜负，胜者进入下一轮，直至决出冠军。
2. 每局比赛限时 5 分钟(决赛10分钟)。
3. 胜利条件：
a) 团灭对手（杀死、打晕或使对方全员离开较技场）
b) 计时结束后存活队员比对方多（晕倒不算存活）
c) 两队全灭或两队存活人数一致时，由主持人根据血量或其他现场情况判定获胜方
d) 存活双方气血百分比相同时，经验低者胜
4. 比赛总时长预计 2 小时，主持人有权视情况调整每局时长
```

注意事项：

```
1. 参赛前 unset wimpy，关掉机器做的逃跑触发，取消休闲玩家设定
2. 禁止外援 – 搬 NPC 进较技场或借助队外玩家帮忙都不允许
3. 禁止使用华山派平沙落雁脚或天魔解体大法
4. 禁止在进入较技场前的准备期以下毒、偷窃、乞讨、下蛊、吹拉弹唱等任何手段攻击或干扰对手
5. 禁止用符和吃药 – 裁判与主持人保留随时检查状态，要求发截图的一切权利
6. 进入较技场后及战斗中允许以下毒、偷窃、乞讨、下蛊、吹拉弹唱等任何手段攻击或干扰对手
7. 战斗前与战斗中允许以放宠物、吹拉弹唱等任何手段增益己方
8. 每局结束主持人宣布胜利方后，双方队伍须：
a) 在 1 分钟内清场并离开（准备好小号随时待命迅速搬走己方尸体）
b) 尽快shichou 对方队伍成员以免误伤
9. 仅允许参战队伍和主持人小号进入较技场。观众请到看台，等待参战的队伍请在较技场外等候。
```

**单人对抗战**

报名要求：

1. 分为 0-10M、11-99M、100-199M、200-399M、400-600M 、600-800M、800-1000M、1000M以上共8组

2. 允许挑战比自己经验更高的组别（比如 80M 报名 100-199M 组），不允许报名低于自身经验的组。每人最多允许报一组

赛制：

```

1.每局比赛限时 3 分钟，总决赛限时5 分钟
2.若对战双方皆为玄武，需要在主持人见证下 remove all（考虑到很多人 perform 完自动 wield shield，佩戴盾牌不算违规）
3.双方进入较技场后，等待主持人在 chat 频道中说出 “比武开始” 后同时向对方下 kill 指令
4.比武初赛采用积分挑战值，每个选手可以有3次机会自由挑战同组选手（不可重复挑战），被挑战者不限制次数，但是积分最高累积9000分。
5.胜利条件：
a) 杀死、打晕或者使对方离开较技场
b) 若计时结束后双方都存活，由主持人根据血量和其他现场情况判定获胜方
c) 若双方都存活且气血百分比相同，经验低者胜
6.比赛总时长预计 2 小时内，主持人有权视情况调整每局时长
```

注意事项：

```
1. 参赛前 unset wimpy，关掉机器做的逃跑触发，取消休闲玩家设定
2. 禁止外援 – 搬 NPC 进较技场或非参战玩家帮忙都不允许
3. 禁止使用华山派平沙落雁脚或天魔解体大法
4. 禁止在进入较技场前的准备期以下毒、偷窃、乞讨、下蛊、吹拉弹唱等任何手段攻击或干扰对手
5. 禁止用符和吃药 – 裁判与主持人保留随时检查状态，要求发截图的一切权利
6. 进入较技场后及战斗中允许以下毒、偷窃、乞讨、下蛊、吹拉弹唱等任何手段攻击或干扰对手
7. 战斗前与战斗中允许以放宠物、吹拉弹唱等任何手段增益自己
8. 每局结束主持人宣布胜利方后，双方须：
a) 在 1 分钟内清场并离开（准备好小号随时待命迅速搬走自己尸体）
b) 尽快shichou 对方以免误伤
9. 仅允许参战选手和主持人小号进入较技场。观众请到看台，等待参战的选手请在较技场外等候。
```

初赛赛制：
萌新比武初赛有宝石拿了！

为激励更多萌新积极报名比武大会，特修改比武初赛规则如下：

比武初赛采用积分挑战值，每个选手可以有3次机会自由挑战同组选手（不可重复挑战），被挑战者不限制次数，但是积分最高累积9000分。

1，挑战胜利者可积分3000分，失败者无积分。

2, 9000积分可兑换泽五行宝石一枚，6000积分兑换火四灵宝石一枚，3000积分兑换火五行宝石一枚。

3，4月10日起，对战双方可以和在线4位裁判其中一位约定比武时间，在裁判监督下进行比武。由裁判记录比赛结果，并录入比武大会赛果里。

决赛赛程：

每组积分前两名或三名（含并列）选手于当日进行决赛，决赛采用三局两胜制，缺席者判负

**多人团队竞技**

~~~~~~~（压阵彩蛋）~~~~~~~

比武人数：多 vs 多 （4人以上自由组合）

比武组别：不限经验，1000M以上玩家不超过4人 (含)
比武赛制：街头混战模式，选出本势力最合适的多人参赛，自行安排战术比赛预计时长：10min以内

比武规则：遭遇战模式：赛前不可对赛场有任何先前布置，裁判chat发令“比赛开始”后，双方集体进入比赛场地开始比赛，
战术自拟。赛前不可对对手施加负面状态，比赛期间不可以擅自离开场地，不许天魔解体，不许吃符吃药, 禁用火药炸药等易燃物品，禁用含沙射影等非常规武器（均由赛委会监督），
其他手段不限。具体规则请参考上文。
