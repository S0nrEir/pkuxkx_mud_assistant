经脉系统

```
在人物等级超过30以后可以打通自己的经脉
```

**主要命令**

| vein | 查看经脉命令系统 |
|---|---|
| vein overall | 查看通脉整体情况 |
| vein list | 获得经脉列表 |
| vein <经脉名> | 了解经脉概况 |
| set vein off | 关闭奇经八脉内力加成 |
| vein through <穴道> | 尝试贯通某经脉，贯通某经脉的命令请注意穴道的正确中文名 |
| vein detail | 目标经脉的通脉情况 |
| vein guide | 通脉七层后，可以引导真气 |
| vein recover | 走火后吃药恢复 |

友情提示：

1、通脉需要最大内力大于一万。真气两层以下无法用真气通脉（六层时试了试可以用了，不过三到五层没试不知道从哪层开始可以，而且用真气通脉消耗的数量和内力一样，set vigour_vein 1来使用真气通脉。PS:五层已测试无法真气通脉，答案是六层才能使用。）

2、vein detail可以看到每天通脉剩余次数（每天有1000次），每次通脉时间，通脉总需时间（你花在通脉上的时间，通脉总时间除以每次通脉时间，就可以估算要通多少次，都是一个很大的数）

3、每次通脉需要内力>100%,先dazuo max，当内力低于100%，无法继续通脉。

4、通脉走火后，1，2层可以找洛阳北丑修复经脉，ask 经脉受损，消耗dep和愿望点。（PS：第一层修复一次要2W dep和2点愿望,3层以上只能吃走火药）

5、7层通脉以后，设置vein，具体设置哪个脉，看你需要的属性。通过vein 经脉名，查看具体引导效果。例如，vein 手太阳小肠经，发现引导效果是增强点穴效果，想要这个效果，则1、set vein 手太阳小肠经 2、vein guide。后续操作，看提示即可。（提示：本条信息未经验证，有大佬7层通脉完成的去试验一下来修改吧）

6、通脉是个时间活，天天挂着通吧骚年！

7、顺比较中庸，获得好处假设是100%，逆通有1/3概率获得200%，也有2/3概率只获得33%，**这里注意逆通并不是说你逆通完成了就叫逆通成功了，需要自己对比通脉前后的效果才知道逆通是否成功。**

8、在冲穴开始的时候，可能出现识图内容，比实际缺少了一个穴道，你需要做的是找到缺少的穴道，用vein enforce <穴道名>来冲破他，
如果冲破了，通脉进度会比正常提升，如果不冲或者忽略，对通脉也没有什么影响。还有更小概率出现图片，直接告诉穴道，同样命令冲穴，会有更大进度提升。这两种识图冲穴都只有在cd大于15秒的经脉才会出现，在vein through的时候出现提示，在内力纳入前必须完成冲穴，否则视为无效。当然无效也不影响你正常通脉。这个难度有点大，那么通了之后，就激活了某条经脉的五行属性。效果在战斗中发挥，效果=平均每件防具的宝石数（如果你都是3o套，那么这个值就是3）×平均质量（比如你平均是雷级宝石）×经脉五行属性。比如你激活了木属性，等于你额外多一件3雷木装备的防御效果。

9、通脉到一定程度会加一个经脉学的skills这玩意没有官方说明，谣传会降低走火。
