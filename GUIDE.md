# MUD 助手使用指南

## 简介

MUD 助手是为 [北大侠客行](https://pkuxkx.net) (mud.pkuxkx.net:5555) 开发的辅助工具集，提供三种使用方式：

| 模式 | 命令 | 说明 |
|------|------|------|
| Web 客户端 | `python start.py web` 或双击 `启动Web客户端.bat` | 浏览器直接玩 MUD，自带命令历史和聊天面板 |
| Telnet 代理 | `python start.py` | 配合 zMUD 使用，自动记录日志和聊天监听 |
| 命令统计 | `python start.py cmdlog` | 查看当天的命令分类统计 |

---

## 一、Web 客户端模式（推荐）

### 启动

双击 `启动Web客户端.bat`，或在命令行中执行：

```
python start.py web
```

启动后会自动打开浏览器访问 `http://127.0.0.1:58080`。

### 界面说明

```
┌──────────────────────────────┬──────────────┐
│  ● 已连接                    │  命令历史     │
│──────────────────────────────│  > look      │
│                              │  > n         │
│                              │  > kill npc  │
│     游戏终端                  │  ...         │
│     (显示服务器返回的内容)     │──────────────│
│     完整 ANSI 颜色支持        │  聊天频道     │
│     可滚动回看 5000 行        │  [14:30] 闲聊 │
│                              │  [14:31] 交易 │
│                              │  ...         │
│──────────────────────────────│              │
│  [输入命令...            ] 发送│              │
└──────────────────────────────┴──────────────┘
```

- **左侧终端**：显示服务器返回的所有内容（房间描述、战斗信息、颜色等）
- **左侧输入框**：在底部输入框中输入命令，按回车或点击"发送"
- **右上**：最近 20 条命令历史，点击任意一条填入输入框
- **右下**：实时聊天消息，自动捕获【闲聊】【谣言】【交易】【QQ群】等频道

### 操作方式

1. 打开页面后看到 MUD 登录画面
2. 在底部输入框输入角色名，回车发送
3. 输入密码，回车发送
4. 正常输入游戏命令即可（look、n、s、kill 等）

### 快捷操作

| 操作 | 说明 |
|------|------|
| 回车 | 发送输入框中的命令 |
| 上方向键 | 输入框中翻阅上一条命令 |
| 下方向键 | 输入框中翻阅下一条命令 |
| 点击终端区域 | 自动聚焦到底部输入框 |
| 点击右侧命令历史 | 填入输入框（可修改后再发送） |

### 注意事项

- 关闭浏览器标签页会断开游戏连接（角色会留在游戏中）
- 可以同时打开多个标签页登录不同角色
- 页面断开后会自动重连
- 服务器地址固定为 `127.0.0.1:58080`，仅本机可访问

## 二、Telnet 代理模式（配合 zMUD）

### 启动

```
python start.py
```

### 连接方式

在 zMUD 中将连接地址改为：

```
地址: 127.0.0.1
端口: 6666
```

代理会自动转发到真实的 MUD 服务器 (mud.pkuxkx.net:5555)。

### 功能

- 所有游戏流量自动记录到 `logs/YYYY-MM-DD_raw.log`
- 启动时会自动弹出一个独立的命令行窗口，专门显示聊天频道消息
- 聊天窗口关闭不影响游戏

### 数据流向

```
zMUD 客户端 ←→ 本代理 (localhost:6666) ←→ MUD 服务器 (mud.pkuxkx.net:5555)
```

## 三、命令统计

查看当天或指定日期的命令分类统计：

```
python start.py cmdlog           # 当天
python start.py cmdlog 2026-06-04  # 指定日期
```

输出 Markdown 格式的统计报告，按分类（移动、战斗、物品、技能、沟通等）列出所有使用过的命令及次数。

统计结果保存在 `cmdlog/YYYY-MM-DD_commands.md`。

## 四、文件说明

```
mud_assistant/
├── 启动Web客户端.bat   # 一键启动 Web MUD 客户端
├── start.py           # 主入口
├── web_mud.py         # Web 客户端（浏览器玩 MUD）
├── proxy.py           # Telnet 代理（配合 zMUD）
├── chat_monitor.py    # 聊天消息过滤模块
├── config.py          # 配置文件
├── skill_cmdlog.py    # 命令分类统计
├── wiki_spider.py     # Wiki 爬虫
├── wiki_clean.py      # Wiki 内容清理
├── logs/              # 原始通信日志
│   └── YYYY-MM-DD_raw.log
└── cmdlog/            # 命令统计报告
    └── YYYY-MM-DD_commands.md
```

## 五、配置

编辑 `config.py` 可修改：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| MUD_HOST | mud.pkuxkx.net | MUD 服务器地址 |
| MUD_PORT | 5555 | MUD 服务器端口 |
| PROXY_HOST | 127.0.0.1 | 代理监听地址 |
| PROXY_PORT | 6666 | 代理监听端口 |
| LOG_DIR | logs | 日志目录 |
| LOG_ENCODING | utf-8 | 日志编码 |

Web 客户端的端口 (58080) 写在 `web_mud.py` 的 `run_server()` 函数中，可直接修改。
