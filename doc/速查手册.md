# 北大侠客行（PKUXKX）新手速查手册

> 基于 wiki 百科整理，供游戏内快速查阅

---

## 一、聊天与消息控制

### 屏蔽聊天频道
- `tune` — 查看当前收听的所有频道
- `tune <频道名>` — 关闭/打开某个频道（开关切换）
- `tune channel helpme` — 示例：关闭求助频道

### 屏蔽战斗信息
- `set skip_combat 1` — 屏蔽普通招式信息
- `set skip_combat 2` — 同时屏蔽回合和互搏提示
- `set skip_combat 3` — 屏蔽除受伤外的所有信息
- `set skip_combat 4` — 屏蔽所有信息（perform除外）

### 其他消息过滤
- `set msgfilter1 *关键词` — 用通配符过滤特定消息
- `set npctalk` — 过滤部分 NPC 在公共频道的发言

## 二、查看状态与信息

### 人物状态
- `hp` — 查看精气、经验等
- `sc` — 查看属性和道德（`sc -xiantian` 查看先天属性）
- `skills` — 查看已学武功技能
- `i` — 查看身上携带物品
- `id here` — 查看当前房间所有物品和人物的 id

### 场景信息
- `look` — 查看当前房间描述
- `look <id>` — 查看具体物品/NPC详情
- `id here` — 查看房间内所有物体 id
- `lm` — 查看当前区域地图（localmaps）
- `lm <地点名>` — 在地图上高亮显示某地点

### 任务查询
- `jq`（jobquery）— 查看当前可做的任务及条件，**不知道干嘛就 jq**

## 三、NPC 交互

### 打听与交流
- `ask <NPC id> about all` — 查看 NPC 知道哪些话题
- `ask <NPC id> about <关键词>` — 向 NPC 打听具体事情
- `id here` — 先用这个查看当前房间 NPC 的 id

### 玩家评论
- `comment here` — 查看当前房间的玩家评论（其他玩家的交互提示）
- `comment <物品/NPC名>` — 查看某物的玩家评论
- `comment -a here` — 添加自己的评论

---

## 四、物品与装备

### 穿脱装备
- `wear <id>` — 穿上衣服/护甲
- `wear all` — 全部穿上
- `remove <id>` — 脱下装备
- `wield <id>` — 装备武器
- `wield all` — 装备所有武器

### 物品操作
- `get <id>` — 捡起物品
- `get <id> from corpse` — 从尸体上拿物品
- `drop <id>` — 丢弃物品
- `i` — 查看身上物品列表（确认 id）

## 五、武功系统

### 学习武功
- `learn <师傅id> for <武功名> <次数>` — 向师傅学武功
- 例：`learn lu guanying for bihai-shengong 10`

### 激发武功（enable / jifa）
- `enable` 或 `jifa` — 查看当前激发状态
- `enable <类型> <武功名>` — 激发武功
  - `enable force bihai-shengong` — 激发特殊内功
  - `enable dodge luoying-shenfa` — 激发轻功
  - `enable parry fuxue-shou` — 激发招架
  - `enable hand fuxue-shou` — 激发手法
  - `enable finger tanzhi-shengong` — 激发指法
  - `enable sword yuxiao-jian` — 激发剑法
- `enable <类型> none` — 取消某项激发

### 准备空手招式（prepare / bei）
- `bei <武功1> <武功2>` — 互备两种空手武功
  - 例：`bei tanzhi-shengong fuxue-shou`
- 有效等级 = 基本武功等级/2 + 特殊武功等级

## 六、移动与地图

### 基本移动
- 方向键：`e/w/s/n/u/d` 及组合（如 `wu`, `ed`）
- `look` 查看房间出口描述中的方向

### 快速移动
- `walk <目的地>` — 系统路径行走
- `walk -c` — 查看目的地列表
- `node` — 查看 node 路径列表（有 ☆ 标记的地点）
- `lm` — 查看当前区域地图
- `lm <地点>` — 在地图上搜索地点

### 门派路引（新手期）
- `travel <目的地>` — 在室外使用，经验 < 5000 且年龄 14 岁有效
- 例：`travel jy` 去晋阳，`travel 归云庄` 去归云庄
- **路引有时效，尽快使用**

## 七、新手流程

### 新手村（柳秀山庄）
- 跟着引导提示完成，不知道做什么就 `jq`
- 最后在游鲲翼处 `changegift` 调整天赋（只能调一次）
- 天赋推荐：**20 20 20 20**（水桶号，万金油）

### 到扬州后必做
1. `pick guo` — 广场大榕树下摘果子（拜师学特殊内功后再吃）
2. `up` 到赏月台，`ask lver about 月饼` — 推荐咸月饼（涨精力）
3. 广场 `wander;w` 找爱爱领免费新手装备（`list` 查看，`draw <id>` 领取）
4. 用路引去门派拜师

### 拜师后
1. 做门派新手任务到 100K 经验（不要先做钓鱼/红豆）
2. 尽早登记门派例银（`help silver` 查看NPC）
3. 24岁前去晋阳洗澡涨根骨
4. 20W 经验前完成新人里程碑

## 八、桃花岛专属

### 拜师
- 入门师傅：**陆冠英**（归云庄）
- 条件：道德 ≥ 0，判师 < 5 次，PK < 10 次，无经验限制
- 拜师命令：`ask lu guanying about 拜师` 或 `apprentice lu guanying`

### 任务
- 新手任务：找**陆乘风**（`lu chengfeng`），`ask lu chengfeng about quest`
- 普通任务：找**王铁**（`wang tie`），同样 ask about quest
- 门忠任务：找**曲灵风**

### 例银
- `help silver` 查看例银 NPC
- 或 `ask lu chengfeng about 例银` 尝试

### 桃花岛核心武功
| 武功 | 类型 | 备注 |
|---|---|---|
| 碧海神功 | 特殊内功 | 必学 |
| 落英身法 | 轻功 | 高闪避 |
| 玉箫剑法 | 剑法 | 主要武器功夫 |
| 弹指神通 | 指法 | 偷袭 pfm，可与兰花互备 |
| 兰花拂穴手 | 手法 | 可与弹指互备 |
| 旋风扫叶腿 | 腿法 | 配合落英身法有反击 |

### 师徒路线
1. **陆冠英**（入门，最多教到 82 级）
2. **陆乘风** / **程英**（碧海神功 30+，最多教到 157 级）
3. **曲灵风**（碧海神功 70+，过渡师傅）
4. **黄药师**（碧海神功 200+，悟性 40，奇门八卦 110+）
